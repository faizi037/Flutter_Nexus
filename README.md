# Flutter_Nexus

This professional-grade Flutter application serves as a comprehensive demonstration of advanced mobile development techniques and core computer science fundamentals. At its heart, the project implements a robust REST API consumption layer, utilizing the Dart http package to fetch, parse, and display real-time data from JSONPlaceholder. By leveraging asynchronous programming patterns and the FutureBuilder pattern, the application ensures a smooth, non-blocking user experience even during intensive network operations.

The UI architecture is designed with scalability in mind, featuring dynamic ListView components with optimized image handling and error recovery states. Beyond the frontend, the project includes a specialized library of Dart algorithm solutions, meticulously crafted to showcase problem-solving skills essential for technical interviews. These implementations cover a wide range of topics, from recursive mathematical series like Fibonacci to fundamental sorting algorithms such as Bubble Sort, built without relying on built-in library functions to demonstrate a deep understanding of logic flows.

Additionally, the project tackles complex "Business Logic" scenarios, such as the 'Candy-Wrapper' problem, which requires precise mathematical modeling and iterative logic. The inclusion of dynamic star patterns and data visualization techniques further highlights the developer's versatility in handling both logic-heavy and UI-centric tasks. This repository is not just a codebase but a portfolio of best practices in Flutter development, including structured project organization, clean code principles, and comprehensive documentation, making it an ideal showcase for prospective employers or collaborators in the Flutter ecosystem.

## Features

- Dynamic Data Fetching: Real-time integration with REST APIs using the http package.
- Asynchronous UI: Efficiently handles network requests with FutureBuilder or state management patterns.
- Image Optimization: Optimized network image loading with error handling.
- Algorithm Showcase: A curated collection of Dart solutions for common interview problems:
  - Recursive Fibonacci series.
  - Bubble Sort implementation (without library functions).
  - Business logic problem solving (Candy Logic).
  - Dynamic pattern generation.

## Tech Stack

- Framework: Flutter
- Language: Dart
- API: JSONPlaceholder
- Networking: http

## Project Structure

- lib/home_screen.dart: Main UI for viewing fetched API data.
- lib/interview_solutions.dart: Core logic for algorithmic solutions.
- lib/main.dart: Application entry point.

## Installation and Setup

1. Clone the repository:
   git clone https://github.com/faizi037/Flutter_Nexus.git

2. Navigate to the project directory:
   cd Flutter_Nexus

3. Get dependencies:
   flutter pub get

4. Run the application:
   flutter run
